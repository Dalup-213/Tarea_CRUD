## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un CRUD (Create, Read, Update, Delete) para la gestión de una Lista de Estudiantes, utilizando el framework Laravel 12 y Tailwind CSS.
El objetivo principal es aplicar los conceptos del patrón MVC (Modelo–Vista–Controlador), reforzando el manejo de rutas, controladores, migraciones, modelos, relaciones y vistas dentro del entorno de Laravel.

El sistema permite registrar, visualizar, editar y eliminar estudiantes desde una base de datos, además de gestionar las carreras mediante un CRUD adicional.

## Objetivos de Aprendizaje

-Reforzar el uso del framework Laravel para desarrollar CRUDs completos.
-Comprender la interacción entre Modelos, Controladores, Vistas y Rutas.
-Practicar la validación de datos y el manejo de formularios.
-Implementar un layout base para organizar las vistas.
-Consolidar la comprensión del flujo completo de un sistema web# Tarea_CRUD
